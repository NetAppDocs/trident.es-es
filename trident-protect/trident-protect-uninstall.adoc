:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs:

[role="lead"]

Para quitar Trident Protect, realice los siguientes pasos.

.Pasos
. Elimine los archivos CR de Trident Protect:
+
[source, console]
----

----
. Quitar Trident Protect:
+
[source, console]
----

----
. Elimine el espacio de nombres Trident Protect:

[source, console]

----
